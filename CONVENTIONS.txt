CLI INTERACTION

Shell interaction should be wrapped in <screen>. Parts typed by the user
should be wrapped in <userinput>

NAMES

Method names, class names, and partial action names should be in
<apiname>. Method names should use () outside <apiname>: <apiname>phpinfo</apiname>()

EXAMPLE CODE

Use <codeblock outputclass="foo"> element where foo can be "php" or "xml". Wrap the
source code in <![CDATA[...]]>

The </codeblock> closing tag must be on the same line as the last line of the code. 

SIDE NOTES

Use <note type="foo"> where type can be "note", "important", "caution" and "tip"

CONCEPTS

Agavi concepts should be wrapped in <keyword>
Foreign concepts should be wrapped in <term>
