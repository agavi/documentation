Installing

You install components to make them available for your solution.

Usually, you can install the components once and then forget about the details of this task. Before installing the components, you should make a plan for what you need to accomplish and the components that will support these goals. After installing the components, you will probably want to perform some preliminary configuration.

  • Installing hard drives or other storage
    A simple guide to installing hard drives.
  • Remove the cover of your system
    This is a simple task requiring a screwdriver.
  • Attach the drive to the system
    The drive needs to be attached to power cables and other cables inside the computer.
  • Replace the cover of your system.
    The cover needs to be put back on to reduce problems from dust.
  • Installing your own Web Server
    Installing a Web Server may sound tricky, but it is quite simple if you follow our instructions.
  • Close all of your programs
    You need to shut down everything else before installing major programs
  • Run Setup.exe
    As with any major program, you will need to run setup in order to install it.
  • Restart your system
    This is standard procedure after installing any new software.
  • Installing your own Database
    Installing a Database may sound tricky, but it is quite simple if you follow our instructions.
